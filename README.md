# ⚔ The Shattered Crown ⚔

A top-down 2D Zelda-like adventure (in the spirit of *A Link to the Past*),
built with Phaser 3. All pixel art and chiptune music are generated by code —
no downloads, works offline.

## How to play

**Online:** play right now at **<https://michaelwales.com/eldoria/>**
(the [player's guide](https://michaelwales.com/eldoria/guide/) is there too).

**Locally:** double-click **`start-game.command`** — it opens the game in your
browser. Leave the little terminal window open while playing.

(Or run `python3 -m http.server 8123` in this folder and open
<http://localhost:8123>.)

> The game is served on a fixed port (8123) so your save files always stick.

## Controls

| Key | Action |
|---|---|
| Arrow keys / WASD | Move |
| SPACE (or Z) | Sword — also talk, read signs, open chests, buy |
| X | Use equipped item (bow, bombs, lantern, potion) |
| C (or TAB) | Switch equipped item |
| ENTER (or ESC / P) | Pause: inventory, equip, **save** |
| M | Mute music/sound |
| E (on title screen) | Erase a save slot (press twice) |

## The quest

Morgrath shattered the Crown of Eldoria into three shards. Find them:

1. 🌲 **Vinewood Temple** in the western forest (the Bow waits inside)
2. 🏜 **Sandstone Ruins** in the southern desert (its entrance has an eye that
   only an arrow can convince... and Bombs wait inside)
3. ❄ **Glacier Hollow** in the northern peaks (a cracked boulder blocks the
   way — boom! — and the Power Boots wait inside)

Three shards dissolve the barrier in the Ashen Wastes. Storm the **Dark
Citadel**. Defeat **Morgrath**. Reforge the crown!

## Secrets (psst...)

- 12 **Golden Acorns** are hidden across the land. A certain squirrel king in
  a hidden forest grove rewards 4, 8 and all 12 of them generously.
- Cracked boulders hide caves. Bombs open them. Some bushes hide stairs...
- A **Sunken Shrine** sleeps somewhere in the swamp. The witch knows more.
  Its treasure doubles your sword damage.
- 3 secret **Heart Containers** are hidden in caves around the world, and
  fairy ponds heal you for free.

## Saving

Three save slots, saved in your browser. The game auto-saves at important
moments (entering areas, finding treasures, beating bosses) and you can save
any time from the pause menu (ENTER → S). Dying never erases progress.
